# Bit && Byte

* Bit = 0-1 bitar (B)
    * Max värde = 1 (0-1)
    * max värde i hexadecimal = 1
    * max värde i binärt = 2^1 -1
* Byte = 8 bits (b)
    * Max värde = 255 (0-255)
    * max värde i hexadecimal = FF
    * max värde i binärt = 2^8 -1

* Word = 16 bits (Intel standard)
    * Max värde = 65535 (0-65535)
    * max värde i hexadecimal = FFFF
    * max värde i binärt = 2^16 -1

* Dword = 32 Bits (Intel standard)
    * Max värde = 4294967295 (0-4294967295)
    * max värde i hexadecimal = FFFFFFFF
    * max värde i binärt = 2^32 -1

* Qword = 64 Bits (Intel standard)
    * Max värde = Mycket.
    * max värde i hexadecimal = FFFFFFFFFFFFFFFF
    * max värde i binärt = 2^64 -1
